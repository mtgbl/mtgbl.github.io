## Commander (Turbo Tower)
[Companion-Code](/faq/#was-hat-es-mit-dem-companion-code-auf-sich): **[GX75MNG](https://magic.wizards.com/en/products/companion-app?shortCode=GX75MNG)**

Die ersten 16 Anmeldungen erhalten eine Command Tower Promokarte! \
Spezialvariante: Jeder Spieler beginnt das Spiel mit der Command Tower Promokarte im Spiel. Die Karte ist nicht Teil des Decks. Das Deck muss immernoch 100 Karten beinhalten. Das Deck darf auch eine andere Kopie von Command Tower enthalten.

Kosten: gratis für Mitglieder, 3.- Fr Startgeld für Nichtmitglieder \
Mitbringen: Commander-Decks; optional: etwas zum Lebenspunkte notieren, Würfel, Tokens, Spielmatte
