# MTGBL Website
Static page generated by Jekyll.

## Requirements
- ruby >=3.2
- gem >=3.4

## Installation
```sh
bundle install
```
## Build
```sh
bundle exec jekyll build
```

## Run locally
```sh
bundle exec jekyll serve
```
Visit at http://127.0.0.1:4000/

## Run locally in Docker
```sh
docker build -t mtgbl . && docker run -it --rm -p 4000:4000 mtgbl
```
Visit at http://127.0.0.1:4000/
